# Portfolio React

Modern portfolio website built with Next.js, MagicUI, and ReactBits components.

## Installation

Install all required packages:

```bash
npm install
```

## Required Packages

The project uses the following main dependencies:

### Core Dependencies
- **next** - React framework for production
- **react** & **react-dom** - React library
- **framer-motion** - Animation library for React
- **lucide-react** - Icon library
- **clsx** - Utility for constructing className strings
- **tailwind-merge** - Merge Tailwind CSS classes

### Dev Dependencies
- **typescript** - TypeScript support
- **tailwindcss** - CSS framework
- **postcss** & **autoprefixer** - CSS processing
- **eslint** - Code linting

## Development

Run the server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Build

Build for production:

```bash
npm run build
npm start
```

## Project Structure

- `src/app/` - Next.js app directory
- `src/components/magicui/` - MagicUI components (Marquee, AnimatedBorder)
- `src/components/reactbits/` - ReactBits components (AnimatedCard, GlowCard)
- `src/components/ui/` - UI components (ProjectCard, TechMarquee)
- `src/lib/` - Utility functions
